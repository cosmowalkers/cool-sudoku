# Changelog

## [0.0.4] - 2026-05-07

### Added

- 设置抽屉菜单（统计页右上角齿轮图标触发）
- 白天/黑夜主题切换
- 状态栏跟随主题变色

### Changed

- 设置项（声音/语言/主题）从统计页平铺改为抽屉菜单集中管理
- 主题系统简化为白天/黑夜两套

### Fixed

- Logo 圆角过大导致安装后超出图标边界
- 黑夜模式下状态栏文字不可见
- React hooks 顺序错误导致崩溃

## [0.0.3] - 2026-05-06

### Changed

- App 名称改为"数独酷"
- 全新 Logo：蓝紫渐变底 + 圆角 9 宫格 + 数字，活泼可爱风格
- 添加 App 描述、iOS bundleIdentifier 等元信息
- Splash 背景色适配设计系统色值

## [0.0.2] - 2026-05-06

### Added

- **微交互动效**: 填数弹跳、错误抖动、行/列/宫完成闪光、数字面板按压反馈、选中格脉冲
- **通关庆祝**: Lottie 撒彩纸动画 + 成绩卡片 + 文字分享
- **音效系统**: 6 个极简音效（填数/错误/撤销/完成行列/通关/成就解锁）+ 静音开关
- **连胜系统**: 每日连胜火焰 + 里程碑庆祝（3/7/14/30/100 天）
- **成就徽章**: 18 个成就定义 + 自动检测 + 解锁弹窗 + 展示网格
- **Reduced motion 支持**: 尊重系统减少动画设置

### Fixed

- 错误反馈不再暴露答案（改为仅在违反数独规则时触发）
- NumberPad 8/9 数字被截断不显示
- 底部 Tab Bar 与游戏内容重叠

### Changed

- 错误计数从"与答案不符"改为"规则冲突次数"
- 连胜数据从每次 history 计算改为直接存储 + persist migrate

## [0.0.1] - 2026-04-30

### Added

- 纯 TypeScript 数独引擎（生成/求解/校验，支持 4 级难度）
- 9×9 棋盘交互（选中高亮、关联格高亮、相同数字高亮、冲突标红）
- 笔记模式（铅笔标记候选数字）
- 操作：撤销 / 擦除 / 提示（每局限 3 次）/ 重来
- 计时器（支持暂停，后台自动暂停）
- 游戏进度自动保存（Zustand + AsyncStorage）
- 统计记录（分难度统计、历史记录）
- 中英双语切换（默认中文）
- 深色模式基础适配
- 触觉反馈
